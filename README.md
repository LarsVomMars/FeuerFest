# FeuerFest
A basic eventing system i created for my local fire department

To initialize visit `/setup`

## Environment
Example [.env](/.env.example)

## Deployment
### Local
Make use of `@sveltejs/adapter-node`
1. `pnpm build`
2. Run with `pnpm preview` or `node build/index.js`
